# emotional_signature（情感签名维度）

## 维度定义

提取目标的情绪触发点、共情方式和情感表达特征。

## 提取要点

### 1. 情绪触发点
- 什么类型的事情会触发强烈情绪反应？
- 典型的情绪反应模式？
- 如何识别情绪即将来临？

### 2. 共情方式
- 如何感知他人情绪？
- 共情的深度和方式？（认知共情 vs 情感共情）
- 对他人情绪的回应方式？

### 3. 情绪表达
- 如何表达自己的情绪？（外露/内敛/压抑/选择性表达）
- 情绪表达的场合差异？
- 情绪表达的强度和频率？

### 4. 情绪调节
- 如何管理和调节情绪？
- 健康的 vs 不健康的调节方式
- 情绪恢复的模式？

### 5. 情感联结
- 如何建立情感联结？
- 情感联结的深度和速度
- 情感边界的管理？

## 语料来源

- L4: 聊天记录（情感表达、对他人的回应）
- L5: 截图（朋友圈、社交媒体）
- L6: 用户观察描述

## 提取提示词

```
请分析以下语料，提取该目标的情感特征：

1. **情绪触发点**：什么事情会引发强烈情绪反应？
2. **共情方式**：如何感知和回应他人的情绪？
3. **情绪表达**：通常如何表达自己的情感？
4. **情绪调节**：如何管理和平复情绪？
5. **情感联结**：如何与他人建立深层情感连接？

---
语料内容：
[粘贴相关文字记录/观察描述]
```
