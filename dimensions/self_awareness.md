# self_awareness（自我认知维度）

## 维度定义

提取目标的自我认知深度、盲点识别能力和反思模式。

## 提取要点

### 1. 自我认知
- 对自身优势和劣势的认知
- 对自身情绪和行为的觉察
- 对自身影响力的认知

### 2. 盲点识别
- 意识不到的思维模式
- 行为中的无意识倾向
- 他人能看到但自己看不到的方面

### 3. 反思模式
- 多久进行一次深度反思？
- 反思的方式和重点？
- 从反思到行动的转化率？

### 4. 反馈接受
- 如何接受负面反馈？
- 主动寻求反馈的程度？
- 对批评的典型反应？

### 5. 自我一致性
- 内在自我和外在表现的一致性
- 不同场合的角色切换
- 真实的自我 vs 呈现的自我

## 语料来源

- L2: 书籍/专栏（自我反思内容）
- L3: 工作文档（复盘、自我总结）
- L4: 聊天记录（分享的自我认知）
- L6: 用户直接描述（"我认为我的优势是..."）

## 提取提示词

```
请分析以下语料，提取该目标的自我认知特征：

1. **自我认知**：对自身能力、性格、影响力的认知程度？
2. **盲点识别**：有哪些自己没意识到但他人能看到的模式？
3. **反思模式**：如何进行自我反思和总结？
4. **反馈接受**：如何面对和处理他人的反馈/批评？
5. **自我一致性**：内在自我和外在表现的一致程度？

---
语料内容：
[粘贴相关文字记录/自我描述]
```
