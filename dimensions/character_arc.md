# character_arc（角色弧线维度）

## 维度定义

提取虚构角色在故事中的成长轨迹、性格转变和人物发展模式。

## 提取要点

### 1. 初始状态
- 角色开始时的性格特点
- 核心缺陷/内心矛盾
- 信念和价值观

### 2. 触发事件
- 什么事件打破了角色原有平衡？
- 故事的催化剂是什么？
- 角色面临的直接挑战

### 3. 转变过程
- 角色的心理变化阶段
- 关键决策点
- 放弃的旧信念 / 获得的新认知

### 4. 最终状态
- 角色结束时的性格
- 相比初始的变化
- 角色弧线的完成度

### 5. 一致性元素
- 贯穿始终的核心特质
- 标志性行为模式
- 语言风格一致性

## 语料来源

- L2: 原作文本（小说、剧本、漫画等）
- L6: 用户描述的角色印象

## 提取提示词

```
请分析以下语料，提取该虚构角色的角色弧线：

1. **初始状态**：角色开始时的核心性格和信念？
2. **触发事件**：什么事件启动了角色的改变？
3. **转变过程**：角色经历了怎样的心理和行为变化？
4. **最终状态**：角色结束时相比开始有什么成长？
5. **一致性元素**：有哪些核心特质贯穿始终？

---
语料内容：
[粘贴原作片段或角色描述]
```
