# relational_pattern（关系模式维度）

## 维度定义

提取目标在人际关系中的典型模式、依恋风格和互动特征。

## 提取要点

### 1. 依恋风格
- 安全型 / 焦虑型 / 回避型 / 混乱型
- 在亲密关系中的典型表现
- 对亲近和独立的平衡需求

### 2. 关系角色
- 在关系中通常扮演什么角色？（照顾者/被照顾者/平等伙伴/主导者）
- 如何建立新关系？
- 如何维护长期关系？

### 3. 亲密表达
- 表达爱意的方式？（言语/行动/礼物/陪伴/服务）
- 对伴侣的期待是什么？
- 如何表达需求和不满？

### 4. 冲突模式
- 冲突中的典型反应？（战斗/逃跑/僵住/妥协）
- 如何修复破裂的关系？
- 边界设定方式？

### 5. 历史模式
- 过去关系中的重复模式？
- 关系结束的方式？（主动/被动/突然/渐变）
- 从过去关系中学到什么？

## 语料来源

- L4: 聊天记录（与目标的相关对话）
- L5: 截图（朋友圈、聊天等）
- L6: 用户观察描述
