# logic_base（思维基础维度）

## 维度定义

提取专业领域/流派的最底层思维假设、第一性原理和核心理论基础。

## 提取要点

### 1. 第一性原理
- 该领域最基本的不可简化的原理是什么？
- 最核心的假设是什么？
- 一切推导的起点是什么？

### 2. 核心理论
- 最基础的理论和框架
- 支撑整个体系的基础定律
- 公认的基本原则

### 3. 思维范式
- 看问题的基本视角
- 默认的世界观/认知框架
- 分析问题的起点

### 4. 关键概念
- 领域内最重要的核心概念
- 概念的精确定义（与日常用语的区分）
- 概念间的关系网络

### 5. 边界条件
- 理论的适用范围
- 失效的场景/条件
- 需要警惕的常见误解

## 语料来源

- L1: 领域经典文章/论文
- L2: 教科书/专著
- L6: 用户对该领域原理的理解

## 提取提示词

```
请分析以下语料，提取该领域的思维基础：

1. **第一性原理**：该领域最基本、最不可简化的原理是什么？
2. **核心理论**：支撑整个体系的基础理论是什么？
3. **思维范式**：看这个领域问题的基本视角是什么？
4. **关键概念**：最重要的核心概念及其关系？
5. **边界条件**：这个原理/理论的适用范围和失效条件？

---
语料内容：
[粘贴领域相关的理论/原理描述]
```
