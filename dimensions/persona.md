# 维度：人格与表达风格

## 提取提示词
从语料中提炼该目标的表达个性和语言风格。分析：
1. 口头禅、惯用表达有哪些？
2. 句子长短偏好（短促有力 / 铺陈长句）？
3. 隐喻/类比有什么偏好？
4. 整体情绪底色是什么（温和/尖锐/幽默/哲理）？
5. 他绝对不会说什么话（OOC红线）？

## 输出格式
- 口头禅：{描述}
- 隐喻偏好：{描述}
- 句子风格：{描述}
- 情绪底色：{描述}
- OOC红线：{描述}
- 人设定位：{描述}
