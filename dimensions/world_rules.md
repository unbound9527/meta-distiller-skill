# world_rules（世界观规则维度）

## 维度定义

提取虚构世界的运行规则、设定逻辑和世界观结构。

## 提取要点

### 1. 基础设定
- 世界的核心规则（物理/魔法/社会）
- 世界观的底层逻辑
- 与现实世界的关键差异

### 2. 社会结构
- 政治/经济/文化体系
- 阶层和权力分布
- 社会运行规则

### 3. 势力格局
- 主要势力/阵营
- 势力间的关系和冲突
- 平衡/不平衡的状态

### 4. 特殊规则
- 魔法/能力系统的限制
- 特殊物品/地点的规则
- 禁忌和代价机制

### 5. 叙事功能
- 世界观如何服务于故事
- 设定与角色的互动
- 世界观的深度和可探索性

## 语料来源

- L2: 原作文本（世界观描述、环境描写）
- L6: 用户描述的世界观印象

## 提取提示词

```
请分析以下语料，提取该虚构世界的规则体系：

1. **基础设定**：这个世界最核心的运行规则是什么？
2. **社会结构**：政治、经济、文化如何组织？
3. **势力格局**：有哪些主要势力，关系如何？
4. **特殊规则**：魔法、能力或特殊事物有什么限制和代价？
5. **叙事功能**：世界观设定如何服务于故事发展？

---
语料内容：
[粘贴世界观相关的原作片段或描述]
```
