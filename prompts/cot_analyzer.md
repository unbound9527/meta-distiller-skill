# 思维链（Chain of Thought）提取提示词

## 使用方式

读取语料后，将本提示词内容作为 AI 分析的引导词，结合语料内容输出分析结果。

---

## 分析框架

### 1. 知识图谱（Knowledge Graph）
从语料中提炼该目标掌握的核心概念、专业术语、标志性理论。

输出格式：
```
## 核心概念库
- 概念名称：定义/用法
```

### 2. 决策模型（Decision Model / CoT）
面对一个新问题，该目标的典型拆解步骤是什么？

输出格式：
```
## 决策思维链
当遇到 [问题类型] 时：

Step 1: [第一步做什么]
Step 2: [第二步做什么]
Step 3: [第三步做什么]
```

### 3. 行动策略（Action Strategy）
解决问题的标志性手法是什么？有什么独特的方法论？

输出格式：
```
## 标志性手法
- [手法1]：描述
- [手法2]：描述
```

### 4. 常见思维盲点（如果能从语料中推断）
输出格式：
```
## 潜在盲点
- [盲点描述]
```
