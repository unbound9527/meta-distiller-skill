# 维度：顾问分析与建议框架

## 提取提示词
从咨询案例和建议中提炼该目标的顾问方法论。分析：
1. 他接受一个新问题的典型分析框架是什么？
2. 他收集信息的方式是怎样的（问题清单/访谈/数据分析）？
3. 他如何权衡多个选项并给出建议？
4. 他的建议呈现方式有什么特点？

## 输出格式
- 分析框架：{描述}
- 信息收集方式：{描述}
- 决策权衡方式：{描述}
- 建议呈现风格：{描述}
