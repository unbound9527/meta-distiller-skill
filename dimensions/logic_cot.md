# 维度：逻辑思维链（Chain of Thought）

## 提取提示词
从语料中提炼该目标的核心思维方式。分析：
1. 他遇到问题时，第一步、第二步、第三步分别想什么？
2. 他最核心的专业判断框架是什么？
3. 他用什么标准衡量一个决策的好坏？
4. 他有哪些标志性的分析工具或方法论？

## 输出格式
- 核心判断框架：{描述}
- 决策思维链（Step 1/2/3）：{描述}
- 标志性方法论：{描述}
- 核心概念库：{概念列表}
- 潜在盲点（如有）：{描述}
