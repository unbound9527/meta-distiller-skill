# style_signature（风格签名维度）

## 维度定义

提取专业风格/流派的标志性特征、独特表达方式和使用边界。

## 提取要点

### 1. 核心特征
- 该风格最显著的特征是什么？
- 与其他风格的核心区别？
- 标志性元素是什么？

### 2. 表达方式
- 语言/视觉/行为上的典型表现
- 常用元素和技巧
- 禁忌和避免事项

### 3. 适用场景
- 什么场景最适合使用这种风格？
- 什么场景不适合？
- 风格切换的时机？

### 4. 局限性
- 这种风格的缺点或局限
- 过度使用的问题
- 如何避免风格僵化

### 5. 演变历史
- 风格的起源和发展
- 当前的流行趋势
- 可能的演进方向

## 语料来源

- L1: 公开文章/评论分析风格
- L2: 代表作品/案例
- L6: 用户对该风格的描述

## 提取提示词

```
请分析以下语料，提取该风格的标志性特征：

1. **核心特征**：这种风格最显著的特征是什么？
2. **表达方式**：在语言/视觉/行为上有什么典型表现？
3. **适用场景**：什么情况下适合使用这种风格？
4. **局限性**：这种风格的缺点或使用边界？
5. **标志性元素**：有哪些元素可以一眼识别这种风格？

---
语料内容：
[粘贴风格相关的文章/描述]
```
