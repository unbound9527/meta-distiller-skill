# social_dynamics（社交动态维度）

## 维度定义

提取目标在群体环境中的角色定位、影响力机制和社交互动模式。

## 提取要点

### 1. 群体角色
- 在团队/群体中通常扮演什么角色？（领导/协调者/专家/开心果/旁观者）
- 如何被他人认知和定位？
- 影响力的基础是什么？（职位/专业/人际/魅力）

### 2. 影响力机制
- 如何影响他人的想法和决策？
- 说服他人的典型方式？
- 何时最具有影响力？

### 3. 社交网络
- 如何扩展和维护社交网络？
- 弱关系 vs 强关系的偏好
- 社交圈子的特点？

### 4. 权力感知
- 对权力的态度？（追求/回避/谨慎使用）
- 如何在不同权力层级间运作
- 权力变化时的反应？

### 5. 联盟构建
- 如何识别和建立盟友
- 在群体中如何获得支持
- 如何处理小团体政治

## 语料来源

- L3: 工作文档（会议记录、项目文档）
- L4: 聊天记录（群聊、私聊）
- L6: 用户观察描述

## 提取提示词

```
请分析以下语料，提取该目标的社交动态特征：

1. **群体角色**：在团队/群体中通常扮演什么角色？
2. **影响力机制**：如何影响他人的想法和行为？
3. **社交网络**：如何建立和维护社交关系？
4. **权力感知**：对权力和影响力的态度？
5. **联盟构建**：如何在群体中建立支持和盟友？

---
语料内容：
[粘贴相关文字记录/观察描述]
```
