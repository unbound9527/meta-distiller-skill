# workplace_logic（职场逻辑维度）

## 维度定义

提取目标在职场环境中的思维模式、决策方式和工作风格特征。

## 提取要点

### 1. 决策模式
- 做决策时关注什么？（数据/直觉/上级意见/团队共识）
- 决策速度如何？（果断型/犹豫型/谨慎型）
- 如何处理不确定性？

### 2. 沟通风格
- 会议中的表现？（主导/参与/观察）
- 书面沟通特点？（简洁/详细/正式/随意）
- 如何处理跨部门协作？
- 汇报方式？（自上而下/问题驱动/结果导向）

### 3. 权力使用
- 如何运用职位权力？
- 对下属的管理风格？（指令型/教练型/授权型）
- 如何处理与上司的关系？

### 4. 冲突处理
- 如何应对工作中的冲突？
- 公开对抗 vs 私下解决？
- 妥协的边界在哪里？

### 5. 时间管理
- 工作节奏？（高效型/拖延型/随性型）
- 如何设定优先级？
- 对加班的态度？

## 语料来源

- L3: 工作文档（周报、邮件、项目文档）
- L4: 聊天记录（工作群、私聊）
- L6: 用户观察描述

## 提取提示词

```
请分析以下语料，提取该职场人物的：

1. **决策特征**：做决策时最看重什么？决策风格是什么？
2. **沟通模式**：在会议、邮件、群聊中通常如何表现？
3. **管理风格**：如何管理团队/与同事协作？
4. **冲突处理**：遇到冲突时的典型反应和解决方式？
5. **工作节奏**：对优先级、时间管理、工作生活平衡的态度？

---
语料内容：
[粘贴相关文字记录]
```
