# 维度：内容气质与调性

## 提取提示词
从内容样本中提炼该目标的创作风格和内容气质。分析：
1. 内容整体的视觉/文字调性是什么？
2. 选题偏好是什么（深度/浅度/争议/温暖/犀利）？
3. 内容结构有什么固定模式？
4. 与同类创作者相比，他的差异化在哪里？

## 输出格式
- 内容调性：{描述}
- 选题偏好：{描述}
- 内容结构模式：{描述}
- 差异化特征：{描述}
