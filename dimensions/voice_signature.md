# 维度：声音签名与独特表达标识

## 提取提示词
从文本或音频样本中提炼该目标的独特表达标识。分析：
1. 有什么标志性的用词或短语只有他会用？
2. 他的句式结构有什么独特之处（倒装/重复/排比）？
3. 他有什么专属的概念命名或术语发明？
4. 听众/读者为什么会觉得"这就是他"？

## 输出格式
- 标志性用词：{列表}
- 句式特征：{描述}
- 概念命名：{列表}
- 身份识别度：{描述}
