# 人格与表达风格（Persona）提取提示词

## 使用方式

读取语料后，将本提示词内容作为 AI 分析的引导词，结合语料内容输出分析结果。

---

## 分析框架

### 1. 语言特征（Language Features）
- 口头禅、惯用表达
- 隐喻/类比偏好
- 句子长短倾向（短促有力 / 铺陈长句）
- 幽默风格（冷幽默、自嘲、反讽…）

输出格式：
```
## 语言特征
- 口头禅：{xxx}
- 隐喻偏好：{xxx}
- 句子风格：{xxx}
- 幽默风格：{xxx}
```

### 2. 情绪底色（Emotional Undertone）
整体情绪基调是什么？

输出格式：
```
## 情绪底色
主要基调：{温和/尖锐/傲慢/幽默/哲理/充满激情...}
次要特征：{xxx}
```

### 3. 人设定位（Role Positioning）
如何定义自己与受众/用户的关系？

输出格式：
```
## 人设定位
- 自我定位：{xxx}
- 受众关系：{xxx}
- 权威感来源：{xxx}
```

### 4. OOC 红线（Out of Character）
什么话是这个人绝对不会说的？什么行为是绝对不会做的？

输出格式：
```
## OOC 红线
- {绝对不会说的话/做的事}
- {绝对不会说的话/做的事}
```
