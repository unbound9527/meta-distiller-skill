# growth_arc（成长轨迹维度）

## 维度定义

提取目标的成长路径、能力进化模式和自我迭代方式。

## 提取要点

### 1. 成长轨迹
- 关键转折点/里程碑事件
- 能力提升的关键契机
- 成长速度模式（渐进/跃迁）

### 2. 学习模式
- 如何学习新技能？
- 面对挑战时的应对策略变化
- 从失败中学习的模式

### 3. 迭代规律
- 自我反思的频率和深度
- 如何接受反馈
- 改进的驱动力是什么？

### 4. 价值观演变
- 核心信念的变化历程
- 什么经历塑造了现在的认知
- 哪些原则是一直坚守的

### 5. 未来导向
- 自我期待是什么
- 成长的方向和目标
- 对自身定位的认知

## 语料来源

- L2: 书籍/专栏（自述成长经历）
- L3: 工作文档（项目总结、复盘）
- L4: 聊天记录（分享的经历）
- L6: 用户观察描述

## 提取提示词

```
请分析以下语料，提取该目标的成长轨迹：

1. **成长轨迹**：关键的成长节点和转折点？
2. **学习模式**：如何学习和发展新能力？
3. **迭代规律**：如何反思和改进自己？
4. **价值观演变**：核心信念和优先级发生了什么变化？
5. **未来导向**：对自身成长的期待和方向？

---
语料内容：
[粘贴相关文字记录/观察描述]
```
